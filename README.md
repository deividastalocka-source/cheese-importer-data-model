The purpose of the project was to design a data model for a company that imports cheese, allowing them to predict the impact of currency exchange rate fluctuations on their profit margins.

Project Contents

Excel File: The main spreadsheet containing the data model.
Sheets Include:

* Product & Sales Data
* Graphs & Charts
* Automated Decision Flow
* Optimized User Interface

Formulas & Features:

* Gross and net profit calculations
* Conditional formatting for visual alerts
* Spinner buttons for easy data input
* Dropdowns for graph selection
* Macro buttons for PDF export


Key Features

* Predicts company profit margins under different exchange rate scenarios.
* Interactive interface for non-expert users.
* Automated decision suggestions (“Increase Profit Margin” vs “Keep Sales Price”).
* Multiple designs and optimizations included to improve usability.
* Visual data analysis with charts and graphs.


How to Use

* Open the Excel file in Microsoft Excel.
* Use the main menu to navigate through sheets.
* Input exchange rate values and adjust profit margins using spinner buttons.
* Review charts and analysis on the Graphs Sheet.
* Save or export results using the provided macro buttons.
